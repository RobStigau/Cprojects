# Cprojects
This project is a C++ Blackjack game modeled after a similar game created by me in Python for my Computer Science Principles class in high school. This project builds upon that by using object-oriented programming and inheritance to streamline the design of the project and make the code more modular. This makes it much easier to add changes in the future. This game does work currently, but there are plans in the future to further improve visuals and gameplay. Some of these goals include:
- Ability for the player to split hands
- Add animation to cards and button presses to make the game more immersive and fluid
- Add sounds to further increase said immersion
- Add a betting system so each hand has more impact and the game has an end goal
